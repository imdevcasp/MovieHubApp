# MovieHub
MovieHub is a movie streaming webapp that uses AutoEmbed, VidLink, VidBinge, & VidSrc v2 to stream the movies so adblock is recomended for the best experience.

## Installation
> Download the MovieHub project and add your own tmdb api key to the app.js file then open the index.html and enjoy.
